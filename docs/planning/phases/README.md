# Phase Specs

One spec file per phase, named `PHASE_N_X_TITLE.md` (e.g., `PHASE_1_BASELINE_TESTS.md`).

Use the template in [`.claude/skills/phase-plan/SKILL.md`](../../../.claude/skills/phase-plan/SKILL.md).

The current phase status is tracked in [`docs/ai-context/AI_06_CURRENT_STATE_VS_FUTURE_STATE.md`](../../ai-context/AI_06_CURRENT_STATE_VS_FUTURE_STATE.md).
